# 🎲Grimório do Mestre

## Descrição

O Grimório do Mestre é uma aplicação Java desenvolvida para auxiliar mestres de RPG na organização de suas campanhas. O sistema permite centralizar informações importantes como:

- NPCs.
- eventos.
- sessões.
- elementos de lore.

facilitando o gerenciamento de histórias complexas.

---

## Funcionalidades

- Criação e gerenciamento de campanhas
- Cadastro de personagens (NPCs e inimigos)
- Registro de sessões (resumo do que aconteceu)
- Armazenamento de lore (história do mundo)
- Relacionamento entre personagens (ex: aliados, inimigos)
- Upload de imagens (personagens, mapas, etc.)
- Linha do tempo da campanha

---

## Tecnologias idealizadas para Utilizar

- Java 25/Swing/JavaFX
- Mysql Workbench
