## 🎵 JMusicBox - Console Music Player

Um player de música totalmente funcional em console Java, seguindo o padrão MVC e com persistência em banco de dados MySQL.

## Funcionalidades
- Pode adicionar suas musicas usando a pasta chamada "Musicas".
- Player de musica funcional
- Criação de playlists

--- 

## Como usar

### 1. primeiro passo
Ao abrir o programa, ele automaticamente:
- 📁 Escaneia a pasta `musicas/`
- 🏷️ Extrai metadados das músicas
- 💾 Salva tudo no banco de dados

### 2. Comandos Básicos

| O que fazer | Digite |
|-------------|--------|
| Tocar música | `play` ou `p` |
| Próxima música | `next` ou `n` |
| Música anterior | `prev` ou `b` |
| Parar música | `stop` ou `s` |
| Ver todas as músicas | `list` ou `l` |
| Ver ajuda | `info` ou `i` |
| Sair | `quit` ou `q` |

### 3. Trabalhando com Playlists

| O que fazer | Digite |
|-------------|--------|
| Ver playlists | `playlists` |
| Criar nova | `criar` |
| Carregar uma | `carregar` |
| Adicionar música | `add` |
| Voltar à biblioteca | `voltar` ou `v` |

## 🎵 JMusicBox - Console Music Player

A fully functional console-based music player in Java, following the MVC pattern with MySQL database persistence.

## Features
- Add your music using the folder named "Musicas"
- Functional music player
- Playlist creation

---

## How to Use

### 1. First Step
When you open the program, it automatically:
- 📁 Scans the `musicas/` folder
- 🏷️ Extracts metadata from songs
- 💾 Saves everything to the database

### 2. Basic Commands

| Action | Type |
|--------|------|
| Play music | `play` or `p` |
| Next song | `next` or `n` |
| Previous song | `prev` or `b` |
| Stop music | `stop` or `s` |
| View all songs | `list` or `l` |
| View help | `info` or `i` |
| Exit | `quit` or `q` |

### 3. Working with Playlists

| Action | Type |
|--------|------|
| View playlists | `playlists` |
| Create new | `criar` |
| Load a playlist | `carregar` |
| Add song | `add` |
| Back to library | `voltar` or `v` |
